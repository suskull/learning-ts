# Module 01: TypeScript Fundamentals & Setup

Welcome to the first module of the TypeScript Mastery Course!

## Learning Objectives
- Understand the benefits of TypeScript over JavaScript.
- Set up a TypeScript development environment.
- Configure the TypeScript compiler (`tsconfig.json`).
- Learn basic types and type inference.

## Exercises
1. **Basic Setup & Types**: Refactor a simple JavaScript function to TypeScript.
